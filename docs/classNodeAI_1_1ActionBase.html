Detailed Description

The base class for Action Nodes.

Action Nodes are the base class for all nodes that perform actions. Actions are performed by the agent and involve some sort of logic that determines the behaviour of the agent or interacts with the environment.

public class MyActionNode : ActionNode
{
public MyActionNode()
{
tooltip = "An Example Action Node";
AddProperty<float>("MyProperty", "My Property", 0.0f);
}

} <br>
}
}

Eval is called when the node is evaluated, and contains all of the logic for the node.

Parameters
agentThe NodeAI_Agent that is currently running the node.
currentThe current node which is being run.
Returns
The current or updated state of the node.
Possible return states When a node is evaluated it may return one of four states based on whether it is currently running, or if it has succeeded.
Running The node is currently running and is still evaluating.
Success The node has succeeded and has finished evaluating.
Failure The node has failed and has finished evaluating.
Idle The node is not currently running and is not evaluating.
