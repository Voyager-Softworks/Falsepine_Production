bool DynamicVaulting.CalculateVaultData ( )
inline

Calculates data related to vaulting.

Returns
True if the character can vault, false otherwise.

This method detects whether there is an object in front of the player which can be vaulted over, and finds the point on the object which can be grabbed by the character.
