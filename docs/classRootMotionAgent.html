Detailed Description

Script integrating root motion movement animation with navmesh agent AI navigation.

This script is meant to be used in conjunction with the NavMeshAgent component. It uses the NavMeshAgent component to control rotation and velocity by rotating the agent and controlling the speed of the root motion animation.
